Solar Irradiance Prediction Using PySpark
======
* Build Various regresssion models and tree models on the North American Mesoscale(NAM) data and solar irradiance data recorded by Georgia Power at UGA.
* Predicted the irradiance of the next 24 hours using 1x1, 3x3, 5x5 grids of NAM data.
* Used Google Cloud Platform(GCP), PySpark, Python, Pvlib, Scikit-learn for the implementation

Cilia Segmentation
======
* Extracted features from the videos of hair like structures on cells called Cilia using Optical Flow and Variance Thresholding.
* Implemented U-net model to perform segmentation on the videos
* Used Google Deep Learning VM, OpenCV, Keras, TensorFlow

Malware Classification
======
* Built various ML algorithms to classify the malware instances on the bytecode files in Microsoft Malware Classification Dataset.
* Used Python, Spark, SparkML and GCP for the classification.
